# wfimport

# Setting Local env

## Setting venv

For Mac:
```
 python3 -m venv venv
 source venv/bin/activate 
 pip3 install -r requirements.txt
```


## Update .env file values
```
HOST=
USER=
PASSWORD=
DATABASE=
...
```

## Run project
```
python3 main.py
```

### To generate requirements.txt
```
pip3 freeze > requirements.txt
```
